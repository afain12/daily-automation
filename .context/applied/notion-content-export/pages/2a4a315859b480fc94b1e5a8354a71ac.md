**Day 2 of Python and LLM Training Reflections**

<summary>
### LLM Fundamentals
- On day two of Python training, learning how LLMs work with context windows
- LLMs become less effective as conversations progress down the chat
- Better to edit earlier messages than continue typing new ones as context degrades
### Effective LLM Interaction
- Providing strict instructions with clear end goals yields better results
- Clear specifications of desired outputs improve performance
- Giving explicit examples of what to look for significantly enhances accuracy
### Web Scraping Project
- Successfully implemented scraping for article names and authors
- Attempted to extract PDF links from the target site
- Initially encountered issues with PDF URL identification
### Debugging Techniques
- LLM generated a debugging file to troubleshoot the issues
- Debugging file revealed where data was being retrieved from
- Helped identify the main pain points in the scraping process
- Key learning: create debugging files when code isn't working and parameters are missing
