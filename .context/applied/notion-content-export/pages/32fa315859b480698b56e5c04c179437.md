### Equipment Under Consideration
**BioFlex System (ANA Testing)**
- Handles autoimmune and infectious disease testing, not allergy testing
- 50 assays run simultaneously vs 6 on Viaplex
- 100 samples per hour throughput vs 50 on Viaplex
- Smaller sample size requirements
- 45 minutes to first result
- Walk-away automation, doesn't require constant monitoring
- Runs full ANA screen automatically, billing only for released results
- Pricing model: cost per reportable result rather than per test run
- Can release selective components (e.g., only SSA/SSB) without releasing full screen
**D100 System (A1C Testing)**
- HPLC methodology (gold standard for A1C)
- 70 samples per hour throughput
- Continuous loading vs batch processing on Seabia
- 2 minutes to first result vs 43 minutes on Seabia
- Minimal maintenance, only column changes needed
- More user-friendly and less labor-intensive than Seabia
- Mayo Clinic chose D100 over Seabia for superior automation
- Comparison pending with TOSO (also HPLC, approximately 80 cents per test)
