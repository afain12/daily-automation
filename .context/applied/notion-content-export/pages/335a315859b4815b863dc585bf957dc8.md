## 📝 How to use this system
**Morning routine (5 min):**
1. Open **Morning Report** view in Master Tasks — scan by workspace group
2. Check **Needs Follow-Up** in Provider CRM — who haven’t you touched recently?
3. Create today's **Daily Standup** from the template below
4. Log activities throughout the day in **Activity Log**
**End of day (2 min):**
- Update **Last Activity** on any Master Task you touched
- Log any calls/meetings in Activity Log
- Update **Last Contact** + **Stage** in Provider CRM if you spoke to a provider
**Friday (10 min):**
- Create a **Weekly Review** from the template
- Review each workspace section honestly
- Set next week’s top 3 priorities

## 🔧 Set up Notion-native features (do this once)
### 1. Add Buttons for quick-logging
In the Notion app, click the **/** command anywhere and type **Button**. Create these:
- **➕ Log Activity** button → action: **Add page to Activity Log** → pre-fill Date = Today
- **➕ New Provider** button → action: **Add page to Provider CRM** → pre-fill Stage = New Lead
- **➕ Daily Standup** button → action: **Add page to Standups & Reviews** → use Daily Standup Template, pre-fill Date = Today, Type = Daily Standup
Place these buttons at the top of this page for one-tap access.
### 2. Set up recurring reminders
In the Notion app:
- Open **Standups & Reviews** database
- Click the **⋯** menu → **Automations**
- Add: **Every weekday at 8:00 AM** → Create page from **Daily Standup Template** with today’s date
- Add: **Every Friday at 4:00 PM** → Create page from **Weekly Review Template** with today’s date
### 3. Add Notion Calendar sync
- Connect Notion Calendar to this workspace
- Meeting notes auto-link to the Meeting Notes DB under Lincoln Reference Laboratory

## 🔗 Database relationship map
**Master Tasks** ← central task hub, filtered by Workspace
	↕ Related Task
**Provider CRM** ← relationship/pipeline tracking
	↕ SAIPA Record
**SAIPA Providers DB** ← reference data (NPI, fax, address, specialty)
	↕ Provider link
**Activity Log** ← timestamped daily actions
	↕ Nestmate Account
**Nestmate Account Tracking** ← account-level tracking
All databases are cross-linked. When you log an activity, you can tie it to a Provider, a Task, and/or a Nestmate Account in one entry.
