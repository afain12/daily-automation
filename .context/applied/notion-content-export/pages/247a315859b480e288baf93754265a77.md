# **In-house LLM potential for quality assurance**
**In-House LLM Concept**
- Build custom LLM for quality assurance and data validation
- Cross-reference data between DOC monitor system and LIS system
- Ensure proper patient information input and correct test ordering
- Reduce accessioning issues and problems
**Implementation Approach**
- Start with brain-numbing QA work as initial use case
- Months of training required as side project
- Task agent running on Lincoln's data infrastructure
- Companion tool for customer service and other departments
**Potential Benefits**
- Early problem detection and tracing
- Automated task assignment to problem lists
- Optimized reporting between sales statistics
- Cost reduction in Pakistan operations
- Reduced human error risk
**Key Questions**
- Quantify potential cost savings in Pakistan
- Measure reduction in human error risk
- Determine training timeline and resource requirements
